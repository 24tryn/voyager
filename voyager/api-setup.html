🗺️ Voyager API Setup

PRIMARY

📍 Mapbox Configuration

Get your token from Mapbox Account. Mapbox powers attractions discovery, geocoding, directions, and interactive maps for Voyager.

📝 Steps:

  1. Go to Mapbox Account Tokens page
  2. Create a new token with access to all public scopes
  3. Default public token is perfect for development
  4. Copy the token and paste below
🔵 Optional: Google APIs Configuration

Google APIs is optional and can be used alongside Mapbox for additional features. Get your API keys from Google Cloud Console.

📝 Steps:

  1. Go to Google Cloud Console
  2. Create a new project
  3. Enable Places API, Maps JavaScript API, Directions API
  4. Create an API key
  5. Paste your key below
